网站编辑

关于这个职位

职位描述：

  1. 1、负责壹财税网站内容的采编及发布；
  2. 2、撰写产品描述文案、产品推广文案，进行产品品牌的宣传推广；
  3. 3、负责微信、微博、论坛运营的策划文案编写，能与用户形成交流，吸纳粉丝。
  4. 4、围绕财务、税务热点话题，制作财税板块资讯；
  5. 5、配合并参与产品策划与运营工作。

任职要求：

  1. 1、具有三年以上网站编辑相关工作经验。
  2. 2、文字功底扎实，思维敏捷，有原创能力，有良好的策划思考能力并能独立撰写方案策划和编写；
  3. 3、对财税行业有极大的热情，熟悉行业动态，擅长财税内容编辑；
  4. 4、对热点事件、新闻动态等信息敏感，及时反应和把握以创作题材和内容；
  5. 5、工作热情高，耐心细致，能够承受较强的工作压力，沟通能力强，团队合作能力强；

职位诱惑

  1. 1. 提供业内具有竞争力的薪酬结构及福利体系：底薪 + 期权 + 年终奖 + 绩优者季度调薪；
  2. 2. 完善的社会保险，六险一金；
  3. 3. 年假5天、各类法定有薪假期；
  4. 4. 部门活动专项经费。
